Intelligent Hyperparameter Optimizer

This project presents a system that dynamically optimizes machine learning hyperparameters based on hardware capabilities, ensuring efficient, stable, and crash-free training.

Problem Statement

Machine learning training becomes inefficient due to improper hyperparameter selection that does not consider system limitations. This leads to crashes, slow convergence, and unstable execution.

Literature Review

Existing AutoML systems optimize accuracy but ignore hardware constraints.

Research Gap

No system dynamically adapts hyperparameters based on device capability.

System Methodology

Dataset

EMNIST/MNIST dataset of 28x28 grayscale digits, normalized and split.

Model

TensorFlow neural network trained with dynamically optimized hyperparameters.

Execution

Training with real-time streaming metrics via WebSockets.
